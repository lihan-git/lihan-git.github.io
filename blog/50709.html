    linux 常用快捷键

    ctrl + N : 创建新文件

    ctrl + S : 保存当前文件

    ctrl + Z : 回退

    ctrl + Tab : 切换浏览器标签页

    ctrl + W : 关闭浏览器标签页

    ctrl + T : 打开浏览器新标签页

    F6 : 切换浏览器窗格

    shift + F10 : 鼠标右键

    super + Tab : 切换运行中的应用

    super + D : 回到桌面

    Alt + space : 调出命令界面

    Alt + F4 : 结束当前应用

    ctrl + Alt + T : 启动终端

    super + L : 锁屏

    super + A : 调出应用菜单

    super + M : 打开系统通知信息

    super + Arrow keys : 捕捉窗口
